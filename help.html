Search the Documentation

Finding Help

You can ask questions and seek help on Stack Overflow with #scalamock tag.

Chat & Scala Forum

ScalaMock does not have its own support chat channel, so we recommend visiting the official Scala Discord in #scala-users.

There is also an official forum which can be found at users.scala-lang.org.

Reporting Bugs

ScalaMock bugs can be submitted to our issues page on github. Please note that this is not the right place to ask for help or general questions about how to use ScalaMock.
